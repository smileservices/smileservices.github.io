Securing Linux Web Servers

Published: Sat 03 November 2018
Updated: Sun 05 July 2020
By vladimir gorea

In Linux.

tags: linux unix security

Workplan:

  1. Change ssh port
  2. Disable password login allow only auth key
  3. Enable firewall

Read about common security rules on digitalocean article

SSH settings are found at /etc/ssh/sshd_config: restart ssh daemon

PasswordAuthentication no
Port {newport}

Restart ssh service

sudo systemctl restart sshd

Enable firewall

sudo ufw enable
sudo ufw allow {ssh_port}
sudo ufw allow 80
